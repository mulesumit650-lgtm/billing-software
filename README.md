Billing Software

This is a simple billing software developed using Python and Tkinter.

## Features
- Enter customer details
- Add product quantities
- Automatic total and tax calculation
- Generate bill in bill area

## Technologies Used
- Python
- Tkinter

## How to Run
1. Install Python
2. Run the file using:
   python billing_software.py
